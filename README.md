# Mexiweb

Velkommen til Los Mexicanacas sin nettside.

Mexiweb er satt opp med rammeverk [Astro](https://docs.astro.build/en/concepts/why-astro/) og bruker [Tailwind CSS](https://tailwindcss.com/) for styling.

## Komme i gang

Sørg for at du har `pnpm` installert. Kjør deretter følgende kommando:

```bash
pnpm install
```

For å starte utviklingsserveren, kjør:

```bash
pnpm dev
```

Dette vil starte en lokal server på `http://localhost:4321`, hvor du kan se nettsiden i utviklingsmodus.
